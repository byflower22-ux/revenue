# 文档中心页面设计

## 概述

在演示模式下，于顶部导航栏新增"文档中心"入口，点击后浏览器新页签打开独立的文档中心页面。页面按迭代版本汇总展示需求文档和流程图，支持目录导航和内容预览。

## 入口

- 位置：顶部导航栏右侧，仅在演示模式下可见
- 交互：点击后 `window.open` 新页签打开
- 样式：按钮形式，带文档图标

## 页面布局

三栏布局：

```
┌──────────┬─────────────────────────────────┐
│          │  [需求文档]  [流程图]            │
│          ├──────────┬──────────────────────┤
│ 迭代版本  │  目录/   │                      │
│ 列表     │  文件列表 │   内容预览区          │
│          │          │                      │
│ V2.5 ←  │  1. xxx   │  markdown渲染/       │
│ V2.4    │  2. xxx   │  流程图全图           │
│ V2.1    │  3. xxx   │                      │
│ V2.0    │          │                      │
└──────────┴──────────┴──────────────────────┘
```

### 左栏：迭代版本列表

- 固定宽度 180px
- 从 `iterations.js` 的 versions 数组读取
- 选中态：蓝色背景 + 左侧蓝色边框
- 默认选中最新迭代

### 顶部 Tab

- 两个选项：**需求文档** / **流程图**
- 切换时中栏和右栏内容联动变化

### 需求文档 Tab

- **中栏**（220px）：解析 markdown 文件的标题结构，生成目录树
  - 支持多级缩进（h1/h2/h3/h4）
  - 点击目录项，右栏滚动定位到对应章节
  - 当前可见章节高亮
- **右栏**：将 markdown 渲染为 HTML 展示
  - 标题带锚点 id，用于目录跳转
  - 基础排版样式（表格、代码块、列表等）

### 流程图 Tab

- **中栏**（220px）：列出当前迭代所有流程图的文件名
  - 选中态：紫色背景 + 左侧紫色边框
  - 显示流程图标题（从文件名提取）
- **右栏**：全图预览选中流程图
  - 支持缩放（滚轮 / 按钮）
  - 支持重置大小
  - 支持下载

### 空状态

- 某迭代无需求文档：中栏和右栏显示"该迭代暂无需求文档"
- 某迭代无流程图：同理

## 文件组织与发现

### 项目目录结构

```
需求文档/
  需求文档-V2.5-调整单订单规则.md
流程图/
  流程图-V2.5-订单规则初始化流程.png
  流程图-V2.5-收入生成主流程.png
```

### 命名约定

文件名中包含版本号（如 `V2.5`）用于自动归属迭代。格式：`{类型}-V{版本}-{描述}.{扩展名}`

### 文档清单配置

`public/doc-manifest.json` 手动维护映射关系：

```json
{
  "versions": [
    {
      "key": "2.5",
      "label": "V2.5 迭代",
      "date": "04-25",
      "docs": [
        { "title": "调整单 — 订单规则模块 PRD", "file": "需求文档-V2.5-调整单订单规则.md" }
      ],
      "flows": [
        { "title": "订单规则初始化流程", "file": "流程图-V2.5-订单规则初始化流程.png" },
        { "title": "收入生成主流程", "file": "流程图-V2.5-收入生成主流程.png" }
      ]
    }
  ]
}
```

文件存放在 `public/docs/` 和 `public/flows/` 下，运行时 fetch 读取。

## 技术选型

- **独立 HTML 页面**（`public/doc-center.html`），不依赖 React 路由
- **Markdown 渲染**：`marked` 库（CDN 引入）
- **流程图缩放**：CSS transform + 滚轮事件
- **样式**：内联 CSS，风格与主系统一致（Ant Design 色系）
- **数据**：fetch `doc-manifest.json` + fetch 具体 .md/.png 文件

## 实现文件清单

| 文件 | 说明 |
|------|------|
| `public/doc-center.html` | 文档中心独立页面 |
| `public/doc-manifest.json` | 文档清单配置 |
| `public/docs/` | 需求文档 .md 文件 |
| `public/flows/` | 流程图 .png 文件 |
| `src/App.jsx` | 顶部导航栏增加入口按钮（演示模式下可见） |
